# Custom Commands - SlimeCrush

Projekt-spezifische Slash-Commands für effizientes Arbeiten.

## Verfügbare Commands

### /run-game

Startet das Spiel über MCP.

```
/run-game
```

### /check-errors

Holt Debug-Output und analysiert Fehler.

```
/check-errors
```

## Geplante Commands

- `/win-level` - Aktiviert Win-Cheat für Testing
- `/reset-board` - Setzt das Spielfeld zurück
- `/export-android` - Startet Android Export

## Command-Struktur

Jeder Command ist eine `.md` Datei mit:
- Beschreibung
- Verwendung
- Implementierung (welche Tools/MCP-Calls)
- Fehlerbehandlung
